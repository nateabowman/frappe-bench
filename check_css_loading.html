CSS Loading Diagnostic

Open your browser's Developer Tools (F12) and check:

  1. Go to the Network tab
  2. Filter by "CSS"
  3. Reload the page (Ctrl+Shift+R or Cmd+Shift+R for hard refresh)
  4. Look for these files:
    • design_system.css
    • components.css
    • construction_modules.css
    • mobile_enhancements.css
    • ignitr_brand.css
  5. Check if they return status 200 (loaded successfully)

If you see orange text above, CSS is working!

In the browser console, run: document.querySelectorAll('link[href*="ignitr_brand"]').length

This should return 5 (one for each CSS file).
